Q1: What is CSS Flexbox?

It's used as a container. It provides a better way to add layouts & align contents.

Q2: How do you enable Flexbox on an element?

To enable Flexbox on an element we need to use display: flex; in an element.

Q3: What is the main axis and cross axis in Flexbox?

Main axis: The main axis(x) is the primary axis which is horizontal when the flex-direction property is set to row, and vertical that is top to bottom if flex-direction is set to column.
justify-content control the alignment of flex items on the main axis.
Cross axis: It's perpendicular to main axis.
align-items and align-content control the alignment of flex items on the cross axis.

Q4: How do you align items along the main axis and cross axis in Flexbox?

justify-content property is used to align items along the main axis.
ex- justify-content: center;
align-items property is used to align items along the cross axis.
ex- align-items: center;

Q6:Justify-content allows you to do what?

Justify-content allows us to perform various operations like:
1. flex-start : Aligns items to the start of the container
2. flex-end: Aligns items to the end of the container
3. center: Aligns items at the center of the container
4. space-between: Adds same amount of space between items
